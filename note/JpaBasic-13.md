# 데이터베이스 스키마 자동 생성

---
* DDL을 애플리케이션 실행 시점에 자동 생성
* 테이블 중심 -> 객체 중심
* 데이터베이스 방언(Dialect)을 활용하여 데이터베이스에 맞는 적절한 DDL 생성
* 이렇게 __생성된 DDL은 '개발'에서만 사용__ : 운영에서 쓰면 큰 사고난다...
* 생성된 DDL은 운영서버에서 사용하지 않거나, 적절히 다듬은 후 사용
---

* *.yml 예시
```yaml
spring:
  ...
  
  jpa:
    hibernate:
      ddl-auto: create, update, none, etc...
```
