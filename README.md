# jpa-basic
jpa 기본

### JPA는 왜 실무에서 적용이 어려울까?
객체와 테이블의 올바른 Mapping + 설계가 필요하다.
이걸 이해하지 못하고 잘못된 설계를 해두면 두고두고 어렵다.

### 주요 포인트
1. 객체와 테이블을 제대로 설계하고 맵핑해야한다.
2. 기본 키와 외래키 맵핑
3. 1:N, N:1, N:M 맵핑
4. 실무 노하우 + 성능 고려

### 목표
1. JPA의 내부 동작 방식 이해
2. JPA가 어떤 SQL을 만들어내는지 이해
3. JPA가 언제 SQL을 실행하는지 이해
